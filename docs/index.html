ディズニーリアルタイム待ち時間LINE-bot

ディズニーランド、ディズニーシーのリアルタイム待ち時間を教えてくれるLINE-bot. LINEのトーク画面で"land"とか"sea"とか送信するとそれぞれのテーマパークのリアルタイムの待ち時間を返してくれる. GCPの無料枠で運用しているため個人用

  • 使用技術
  • Node.js (確か12...)
  • GoogleCloudFunctions
  • LINE Message API

水道光熱費登録LINE-bot

水道光熱費をGoogleDrive上のエクセルに登録するLINE-bot. GoogleDrive上に配置したスクリプトをLINE-botのWebHookにした（気がする）。これも個人的には結構便利で、シンプルなフォーマットで登録が可能. CRUD処理はできるようにしないとなぁ…

  • 使用技術
  • GoogleAppScript
  • GoogleCloudFunctions
  • LINE Message API

Okojotter

自分用のつぶやき専用Twitterクライアント. 昔く○いろいどというアプリを使っていたが、最新バージョンに対応しきれていなかったので自分で作成. 複数アカウント登録とか 、複数アカウントからのつぶやきとか、hash tagの保存とか、実況用に必要な機能はとりあえず追加した. ホームボタンのアシストアプリに設定できるようにしてるので便利. お気に入りなのでメンテしていきたい

  • 使用技術
  • Kotlin
  • Twitter4j
